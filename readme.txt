Wordpress plugin for xnbot.io chatbots

== Installation ==

1. Upload `xnbot` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Connect Xnbot.io to Your site
